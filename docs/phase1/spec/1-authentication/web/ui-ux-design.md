# AI Trainer - UI/UX設計書

## 1. デザインコンセプト

### 1.1 デザインの方向性
- **親しみやすさ**: Duolingo風の可愛いキャラクターで毎日触りたくなる
- **シンプルさ**: 複雑な機能は隠し、必要な情報だけを表示
- **達成感**: 進捗が視覚的にわかりやすく、モチベーション向上
- **ポジティブ**: 励ましのメッセージ、成長の喜び

### 1.2 キーワード
- 成長
- 継続
- 励まし
- わかりやすい
- 楽しい

### 1.3 ターゲットユーザー像
- **年齢**: 20代〜40代
- **属性**: 社会人、学生
- **目的**: 資格取得、スキルアップ、自己啓発
- **課題**: 計画が苦手、三日坊主になりやすい

## 3. 画面設計

### 3.1 画面一覧

| 画面名 | パス | 認証 | 説明 |
|--------|------|------|------|
| ログイン | `/login` | 不要 | Google OAuth ログイン |
| ダッシュボード | `/` | 必要 | メイン画面 |
| 目標作成 | `/goal/new` | 必要 | 新規目標作成 |

| カレンダー | `/progress/calendar` | 必要 | カレンダービュー |
| キャラクター | `/character` | 必要 | キャラクター詳細 |
| 設定 | `/settings` | 必要 | ユーザー設定 |

#### コンポーネント

##### 1. キャラクター表示カード
- **サイズ**: 大きめ（目立つように）
- **要素**:
  - キャラクターイラスト（中央）
  - レベル表示（Level 5）
  - 星による視覚化（⭐×レベル数）
  - 経験値バー（250 / 360）
  - 次のレベルまでの経験値
- **アニメーション**:
  - ホバー時に少し拡大
  - レベルアップ時にキラキラエフェクト

##### 2. アクティブゴールカード
- **要素**:
  - 目標タイトル
  - 進捗バー（視覚的に大きく）
  - 実績学習時間 / 推定学習時間
  - パーセンテージ
- **カラー**:
  - 進捗バー: グラデーション（プライマリ → セカンダリ）

##### 3. 次のマイルストーンカード
- **要素**:
  - マイルストーンタイトル
  - 期限日
  - 残り日数（カウントダウン）
- **カラー**:
  - 期限が近い場合: 警告色（オレンジ）

##### 4. 週次サマリーカード
- **要素**:
  - 今週の合計学習時間
  - 記録した日数
  - 連続記録日数（🔥アイコン付き）
- **レイアウト**: 横並び3列

##### 5. CTAボタン
- **テキスト**: 「今日の進捗を記録する」
- **サイズ**: 大きめ、目立つ
- **カラー**: プライマリカラー
- **アニメーション**: ホバー時に少し上に移動

## 4. コンポーネント設計

### 4.1 ボタン (Button)

#### バリエーション
- **Primary**: 主要アクション（プライマリカラー）
- **Secondary**: 副次的アクション（グレー）
- **Danger**: 削除など（赤）
- **Ghost**: テキストのみ

#### サイズ
- **Small**: 32px height
- **Medium**: 40px height (デフォルト)
- **Large**: 48px height

#### 使用例
```tsx
<Button variant="primary" size="large">
  今日の進捗を記録
</Button>
```

---

### 4.2 カード (Card)

#### スタイル
- **背景**: White
- **ボーダー**: Gray-200
- **シャドウ**: subtle shadow
- **角丸**: 8px

#### 使用例
```tsx
<Card>
  <CardHeader>
    <h3>Active Goal</h3>
  </CardHeader>
  <CardBody>
    <p>TOEIC 800点取得</p>
    <ProgressBar value={45} />
  </CardBody>
</Card>
```

---

### 4.3 進捗バー (ProgressBar)

#### スタイル
- **高さ**: 12px
- **背景**: Gray-200
- **前景**: グラデーション (Indigo → Violet)
- **角丸**: 6px

#### 使用例
```tsx
<ProgressBar value={45} max={100} />
```

---

### 4.4 モーダル (Modal)

#### スタイル
- **背景オーバーレイ**: rgba(0, 0, 0, 0.5)
- **モーダル**: White, centered
- **アニメーション**: fade in + scale

#### 使用例
```tsx
<Modal isOpen={isOpen} onClose={handleClose}>
  <ModalHeader>進捗を削除しますか？</ModalHeader>
  <ModalBody>
    <p>この操作は取り消せません。</p>
  </ModalBody>
  <ModalFooter>
    <Button variant="secondary" onClick={handleClose}>
      キャンセル
    </Button>
    <Button variant="danger" onClick={handleDelete}>
      削除
    </Button>
  </ModalFooter>
</Modal>
```

## 7. アニメーション仕様

### 7.1 キャラクターアニメーション

#### 進捗入力時
- **トリガー**: 進捗保存成功
- **アニメーション**:
  - キャラクターがジャンプ（bounce）
  - 喜びの表情
  - キラキラエフェクト（パーティクル）
- **所要時間**: 1秒

#### レベルアップ時
- **トリガー**: 経験値がレベルアップ閾値を超える
- **アニメーション**:
  - 画面全体に光のエフェクト
  - キャラクターが大きく成長（scale up）
  - 「Level Up!」テキスト表示
  - 新しい外見に変化
- **所要時間**: 2秒

#### 中間ゴール達成時
- **トリガー**: マイルストーン達成
- **アニメーション**:
  - 紙吹雪エフェクト（confetti）
  - 花火エフェクト
  - 「おめでとう！」メッセージ
- **所要時間**: 3秒
