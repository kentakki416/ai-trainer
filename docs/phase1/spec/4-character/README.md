# キャラクター成長機能

## コンセプト
Duolingo 風の親しみやすいAIキャラクター「トレちゃん」が日々の学習で成長。ゴール達成でカラー・アクセサリーを解放し、継続モチベーションを高める。

## 成長要素
### レベルシステム
- 経験値計算: `経験値 = 学習時間(分) × 1.5`
- レベルアップ条件: 累積経験値で判定
- レベル上限: MVP は 50、将来的に 100 まで拡張

### 進化段階（appearance_stage）
- Stage1: レベル 1-10（普通のロボット）
- Stage2: 11-25（装飾が追加）
- Stage3: 26-50（大きさが変化）
- Stage4: 51+（最強のロボット）

### カラーバリエーション（appearance_color）
- 初期: ブルー
- ゴール1個達成: ゴールド
- ゴール2個達成: シルバー
- ゴール3個達成: レインボー

### アクセサリー（appearance_accessory）
- ゴール1個達成: リボン
- ゴール2個達成: メガネ
- ゴール3個達成: 王冠

## アニメーション
- 日々の入力完了: 喜びアニメーション
- レベルアップ: 専用エフェクト
- 進化時: 特別演出
- 中間ゴール達成: 花火や紙吹雪
- メインゴール達成: 最高演出 + カラー/アクセサリー解放

## 今後の拡張

    - ユーザーは「トレちゃん」「マスター」など複数のキャラを保持する可能性あり
