# 進捗管理・可視化機能

## カレンダービュー
- 月次カレンダーで日々の進捗を色分け表示
  - 🟢 目標超過
  - 🟡 目標の50%以上
  - ⚪ 未入力
- 日付クリックで詳細記録を表示・編集
- React Day Picker などで実装しレスポンシブ対応

## ダッシュボード（ホーム画面）
1. **キャラクター表示**: レベル、経験値バー、ステージ/カラー
2. **目標カード**: タイトル、達成希望日、累計学習時間
3. **今日のタスク**: AI 提案タスク一覧と完了チェック
4. **中間ゴールカード**: 次に達成すべきゴールの進捗バー
5. **カレンダービュー**: 今月の進捗一覧
6. **進捗入力ボタン**: 画面下部固定で簡易入力モーダルに遷移
