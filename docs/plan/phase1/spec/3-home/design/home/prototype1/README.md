# AI Trainer - HTMLプロトタイプ

## 概要

ホーム画面の静的HTMLプロトタイプです。実際にブラウザで開いて、デザインやレイアウトを確認できます。

## ファイル

- `home.html` - ホーム画面プロトタイプ

## 使い方

### 1. ブラウザで開く

```bash
# プロジェクトルートから
open docs/plan/prototype/home.html

# または、ブラウザで直接ファイルを開く
```

### 2. 動作確認

以下の機能が実装されています:

#### ✅ 実装済み機能

1. **キャラクター表示**:
   - レベル、経験値バー
   - アニメーション（フロート効果）

2. **目標カード**:
   - 目標タイトル、達成希望日
   - 累計学習時間と進捗バー

3. **中間ゴールカード**:
   - 達成済み/進行中/未着手の表示
   - 進捗バー

4. **カレンダー**:
   - 月次表示
   - 学習済み日の✓表示
   - 今日の強調表示

5. **進捗入力ボタン**（画面下部固定）:
   - タップでモーダル表示

6. **進捗入力モーダル**:
   - 簡易入力モード（ワンタップ）
   - 詳細入力モード（切り替え可能）
   - 達成度選択（絵文字）
   - 保存後の成功メッセージ

7. **レスポンシブデザイン**:
   - モバイル、タブレット、デスクトップ対応

#### ⚡ インタラクション

- 「今日の進捗を記録する」ボタン → モーダル表示
- 簡易入力/詳細入力の切り替え
- 達成度の選択
- 保存 → 成功メッセージ表示 → 自動クローズ
- ESCキーでモーダルを閉じる
- モーダル外クリックで閉じる

## デザイン仕様

### カラースキーム

- **プライマリ**: #4A90E2（ブルー）
- **成功**: #7ED321（グリーン）
- **背景**: #F7F9FC（ライトグレー）
- **カード**: #FFFFFF（ホワイト）

### フォント

- システムフォント（-apple-system, BlinkMacSystemFont, Segoe UI, etc.）

### レイアウト

- 最大幅: 800px（デスクトップ）
- カード間隔: 20px
- カード内パディング: 20px
- レスポンシブブレークポイント: 768px
